## Featured System

### AI Audio Translation & Narration Pipeline
End-to-end system that transforms long-form video/audio into translated, synchronized output.

**Pipeline**
- Extract audio → Speech-to-Text → Translation → Text-to-Speech → Merge with original media  
- Preserves timing, background sounds, and speaker flow  

**Stack**
- Azure Speech (STT + TTS), Azure Translator  
- Python processing pipeline  
- Blob Storage + batch orchestration  

**Engineering Challenges Solved**
- Word-level timestamp alignment across languages  
- Maintaining sync between translated speech and original video  
- Handling long-duration media reliably
